## About

Computer Science student at **42 Beirut** and the **University of Science and Arts in Lebanon**.
My focus lies in low-level systems, performance-sensitive code, and the quiet discipline of
building reliable software from first principles — memory, concurrency, networking, and the
layers most developers never see.

I care about clean architecture, thoughtful engineering, and code that reads as clearly as it runs.

<br/>

## Focus

- **Systems Programming** — memory management, process control, and Unix internals in C
- **C++ Engineering** — object-oriented design, RAII, and modern C++ patterns
- **Networking & Protocols** — sockets, HTTP, and non-blocking I/O
- **Algorithms & Data Structures** — deliberate problem-solving and complexity awareness
